Verification Flow

  1. 1 Locate the record binding checksum from the last page of the exported artifact or QR link.
  2. 2 Run the record check here. For a digital PDF file check, select the PDF so the browser can compare its byte hash without uploading the file contents.
  3. 3 Distinguish a record match from a file-bytes match. Stored cryptographic metadata appears only when the artifact response contains a complete, consistent field set, and this browser does not cryptographically revalidate it.
